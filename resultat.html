💡 À propos du Compte de Résultat

Le compte de résultat permet de déterminer si l'entreprise a réalisé un bénéfice ou une perte sur l'exercice.

Formule de calcul :

RÉSULTAT = PRODUITS - CHARGES
  • Si Résultat > 0 : Bénéfice (l'entreprise a gagné de l'argent)
  • Si Résultat < 0 : Perte (l'entreprise a perdu de l'argent)
  • Si Résultat = 0 : Équilibre
